# 🛡️ CyberGuard: Predictive Threat Intelligence System
**Automated Risk Assessment & SQL-Driven Security Analytics**

## 📌 Project Overview
Modern Security Operations Centers (SOC) face "alert fatigue" due to the massive volume of logs. **CyberGuard** is a solution that integrates SQL database management with Machine Learning to prioritize critical threats and automate the initial response.

## 🚀 Key Features
*   **SQL Log Management:** Structured 10,000+ security logs into a relational database for high-performance querying.
*   **Predictive Threat Detection:** Trained a model to identify "Critical Threats" based on temporal patterns and attack types.
*   **Real-Time Risk Scoring Engine:** A Python-based tool that evaluates incoming IP traffic, assigns a risk score, and suggests immediate actions (Block, Flag, or Monitor).
*   **Temporal Danger Zones:** Identified peak attack windows using SQL-driven heatmaps.

## 🛠️ Tech Stack
*   **Database:** SQL (SQLite)
*   **Analysis:** Python (Pandas, NumPy)
*   **Machine Learning:** Scikit-Learn (Random Forest)
*   **Visualization:** Seaborn, Matplotlib
