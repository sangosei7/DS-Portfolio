# 🦖 PokeGuard: Competitive Strategy & Classification System
**Advanced Feature Engineering & Tactical Modeling**

## 📌 Project Overview
This project demonstrates an end-to-end Data Science workflow, moving from messy data extraction to advanced tactical modeling. It transforms raw Pokémon stats into a decision-making tool for competitive gaming.

## 🚀 Key Features
*   **Advanced Data Wrangling:** Extracted structured numeric stats from complex, unformatted text strings using custom parsing functions.
*   **Feature Engineering (The "Pro" Touch):** Created synthetic features like the **Tankiness Index** and **Atk/Def Ratio** to quantify combat roles.
*   **Competitive Tiering Engine:** Developed a prescriptive scoring system that ranks Pokémon into tiers (S, A, B) based on their combat efficiency.
*   **Multiclass Classification:** Implemented a Random Forest model to predict primary types based on physical and combat attributes.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
